# 🏦 ATM Interface - Java

## 📌 Overview
Console-based ATM Interface using Java OOP with 5 classes.

## 🚀 Features
- Secure Login (User ID + PIN)
- Check Balance
- Deposit & Withdraw
- Transfer Money
- Transaction History
- Change PIN

## 📁 Classes
- Main.java, Bank.java, Account.java, ATM.java, Transaction.java

## ▶️ Run
Open in IntelliJ → Run Main.java
